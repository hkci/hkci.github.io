# 香港中央圖書館地圖特藏

香港中央圖書館設立地圖圖書館，專門收藏各式地圖以及與地圖相關的資訊。

  • 主題地圖
  • 衛星地圖
  • 歷史地圖
  • 政府部門繪製的各類香港地圖
    • 地形圖、郊區地圖、街道命名圖（地政總署繪製）
    • 地質圖（土木工程拓展署繪製）
    • 海圖（海事處繪製）
    • 分區計劃大綱圖（規劃署繪製）
