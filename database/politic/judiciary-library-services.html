# 司法機構圖書館 Judiciary Library Services

香港司法機構包括終審法院、高等法院、區域法院，以及各區的裁判法院和審裁處，其中終審法院、高等法院和區域法院設有參考圖書館，分別收藏11,000冊、72,000冊和9200冊資料。內庭、裁判法院及審裁處也有少量藏書。

司法機構圖書館藏普通法相關的資料，類型包括：

  • 法律書籍和期刊
  • 法律彙報和政府憲報
  • 法例和法規
  • 判案書（高等法院藏，1946年至1946年及1966年起，121,000份）
  • 樞密院上訴判案書（1997年以前，182份）
  • 珍本特藏（600多冊）

司法機構圖書館備有電子資源供讀者進館使用，清單詳見：bit.ly/3aLpwUM。
